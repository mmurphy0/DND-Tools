# DND Dice Roller

## Description
A Python Tkinter which lets users virtually roll their dice for actions in Dungeons & Dragons without having to buy dice

## Usage
-> User is welcomed with the home page which has the different types of dice options as button

-> When a button is clicked, another window will pop up with the rolled value

-> User can then click the back button to return to the home page

## Installation
git clone (repo)

cd (repo)

python dnddiceroller.py

or

python3 dnddiceroller.py (depending on operating system)

## Tech Stack
-> Python

-> Tkinter
